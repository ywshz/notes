如何创建子站

有了主站,如 ywshz.github.io后, 一般我们还会想要子站,如ywshz.github.io/notes, 那么该如何实现呢?

其实github pages也为我们提供了这个功能, 只不过稍微要麻烦一点点, 要先建有个repo,再在该repo上新建一个名为gh-pages的分支, 这个分支根目录下要有index.html, 这样就可以了.

  • clone并创建分支并提交到分支

    git clone https://github.com/ywshz/notes.git notes cd notes git checkout --orphan gh-pages git rm -rf . 如果有必要的话 git add . git commit -a -m "First pages commit" git push origin gh-pages

  • 大功告成

    可以通过这样类型的url访问了: ywshz.github.io/notes
